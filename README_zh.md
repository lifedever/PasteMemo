## 功能特色

- **剪贴板历史** — 自动记录文本、链接、图片、代码和文件
- **快速粘贴** — 按 ⌘⇧V 即刻搜索并粘贴历史记录，无需离开当前应用
- **智能识别** — 自动识别内容类型：链接自动抓取图标、代码片段、颜色值、文件路径
- **置顶与收藏** — 置顶重要内容，收藏常用条目，随时快速访问
- **多语言支持** — 英语、中文、日语、韩语、德语、法语、西班牙语
- **菜单栏常驻** — 仅驻留菜单栏，不占用 Dock，资源占用极低
- **自动更新** — 自动检查新版本

## 系统要求

- macOS 14 (Sonoma) 或更高版本
- Apple Silicon 或 Intel Mac

## 安装

### 下载

从 [Releases](https://github.com/lifedever/PasteMemo/releases) 下载最新 `.dmg`：

| 文件 | 架构 |
|------|------|
| `PasteMemo-x.x.x-arm64.dmg` | Apple Silicon (M1/M2/M3/M4) |
| `PasteMemo-x.x.x-x86_64.dmg` | Intel Mac |

> 首次打开时：**右键点击 PasteMemo.app → 打开 → 打开**
>
> 或在终端执行：`xattr -cr /Applications/PasteMemo.app`
